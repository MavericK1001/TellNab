# TellNab Product To-Do Roadmap

## USP Target

**From dilemma to decision with honest threads and strong moderation.**

## Now (0-30 days)

- [x] Dynamic profile identity (no static name for all users)
- [x] Homepage live metrics and featured highlights
- [x] Homepage animation layer (reveal + micro-motion)
- [x] Ask page connected to real advice submission flow
- [ ] Thread reply nesting UI with clear parent/child interactions
- [ ] Better moderation reason visibility for users

## Next (30-90 days)

- [ ] Notifications center (replies, moderation updates, messages)
- [ ] Saved/followed thread watchlist
- [ ] Helpful vote + advisor quality score
- [ ] Messaging user search (remove manual recipient ID requirement)
- [ ] Feed pagination and richer filters

## Later (90+ days)

- [ ] Outcome check-ins (7/30 day updates from original asker)
- [ ] Advisor credibility by category
- [ ] AI writing coach before posting (clarity + civility prompts)
- [ ] AI moderation copilot for human moderators

## Success Metrics

- Advice-to-comment conversion rate
- % threads receiving accepted or marked-useful outcomes
- Repeat 30-day active members
- Moderation resolution time
- Featured-thread engagement uplift
